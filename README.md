## GothamFolio

Life changes from time to time. I've come to a next turn achieving natural limits in my current position and company.

Doing same things day after day with no or very little progress or creativity does not sound right.

Here comes the need for a personal portfolio Web site to display the skills, knowledge and experteese I am about to bring with me to a new employee.

GothamFolio is a personal portfolio Web site reflecting skills and portfolios.

The site is built with the current Laravel (version 11 at the time of start) and Blade with some of Alipnejs at the front-end.
